A linguagem Python

Bom, de todas as linguagens de programação, o python é considerada a mais simples e prática para iniciantes, ela possui bibliotecas incríveis que fazem o python ser ainda mais útil.

O python possui uma sintaxe símples, o que ajuda e muito os iniciantes, é sobre essa linguagem maravilhosa que estarei ensinando meus conhecimentos para você, de forma simples, resumida, e com projetos incríveis, como, um algoritmo que valida cpf, um tipo de spotify, um algoritmo para de login simples, e muitos outros!
